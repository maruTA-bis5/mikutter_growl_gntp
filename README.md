mikutter_growl_gntp
===================

show mikutter notifications via growl using ruby_gntp.


なんぞこれ
----------

mikutterの通知をGrowlにGNTP(Growl Notification Transport Protocol)を用いて送信するプラグインです。


依存関係
--------

ruby_gntpが必要です。gem install ruby_gntp
